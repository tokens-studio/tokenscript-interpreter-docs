# CLI Overview

The TokenScript CLI provides command-line tools for working with design tokens.

## Quick Start

Install globally:

```bash
npm install -g @tokens-studio/tokenscript-interpreter
```

Or use with npx:

```bash
npx tokenscript --help
```
